Reading Python Tracebacks
=========================

By Trey Hunner

Making the slides
-----------------

The slides use hieroglyph.  Install it using:

.. code-block:: bash

    $ pip install hieroglyph

Run the following to make the slides:

.. code-block:: bash

    $ make slides
